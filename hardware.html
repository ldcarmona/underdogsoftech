The hardware fundamentals

In this course you will learn about how all the components in a computer function.
Each individual role in the PC's components, and how they interact amongst each other.

8 steps to creating your Perfect build

- Figure out what's your budget, and what you need this computer for.
- Gather information about the components you need, make sure they are compatible.
Good websites that sell PC parts are; Newegg, Tigerdirect, Micro Center.
- Once you have gathered all your components, start connecting them. Install your CPU first, then the RAM, then the GPU if you have one.
After you have connected all of the components together on the motherboard, place the motherboard inside the case.
- The last component you need to install is the power supply, when installed turn on the hardware and make sure all the components are functioning.
- Once you notice all components are running correctly, for example some good signs are fans spinning, LEDs turning on and so on.
- The final part is selecting which Operating System you will install.
